# Manual setup for Desktop

## 1. DrivenByMoss Bitwig Studio Extension

1. Download and install the appropriate version of [DrivenByMoss](https://www.mossgrabers.de/Software/Bitwig/Bitwig.html) for Bitwig Studio.
